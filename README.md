Log Scaner Utility 1.0. Программа для анализа логов с сервера на вредоносную активность. 

    Производится анализ как одного файла лога, так и нескольких в отдельной папке. Принцип основан на анализе данных клиента, который загружает файл на сервер.
    Для проверки, из лога извлекается IP адрес клиента, время загрузки и вычисленный заранее SHA256 хэш файла.
    Проверка на вредоносную активность осуществляется посредством взаимодействия с Virus Total API v3.
    Результаты проверки формируются в виде файла отчета для каждого файла лога. На выбор можно получать отчет в формате CSV или JSON.
    В отчете реализована запись общих счетчиков срабатываний антивирусных систем по следующим категориям:

    - Malicious (Вредоносные)
    - Suspicious (Подозрительные)
    - Undetected (Неопределенные)
    - Harmless (Безвредные)
    
    Также программа ведет свой системный лог о процессе работы.

    Дополнительно реализован минимальный (тестовый) веб-интерфейс. В качестве клиента выступает браузер, осуществляется проверка одного файла лога.
    В результате браузер получает ответ в json формате.
